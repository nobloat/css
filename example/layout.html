Responsive Layouts

We use the very great idea of Heydon Pickering called the The Flexbox Holy Albatross Reincarnated.

  • no col-sm-1 col-lg-6 col-xl-12 madness
  • no media queries and arbitrary breakoints, which will never cover all device sizes
  • embracing flexbox
  • just use the <section> tag to get resonsiveness
  • configurable margin using <section style="--margin:2rem;"></section>
  • no half fourth-1000 for unequal sized items, just use flex-grow: 2;
  • To wrap items earlier, set the --wrap-at vraiable on the section

Warning: Don't wrap sections in sections, this will result in horizontal scroll bars on mobile devices
